# Tech Context

## Stack
- Site estático com estrutura de tema (Ruby) e dependência de `jekyll` (ver `meu-tema.gemspec`).
- Templates/partials em Liquid.
- CSS principal pré-compilado em `assets/css/index.*.css` (utilitários estilo Tailwind).

## Local dev (quando necessário)
- Rodar via Jekyll (conforme `README.md`): `bundle exec jekyll serve`.
