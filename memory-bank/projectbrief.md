# Project Brief

## Overview
Repositório do site pessoal `pepemesquita.github.io`, publicado via GitHub Pages.

## Goals
- Manter páginas estáticas (ex.: `about.html`, `project.html`) com bom layout/UX.
- Facilitar manutenção do conteúdo via templates e dados (ex.: `_data/projects.json`).

## Non-goals
- Não é um app backend; sem API própria.
- Evitar dependência de build local para mudanças simples de layout (preferir CSS/HTML direto quando possível).
